# Task Plan

## Task 1: Update App.tsx to correctly toggle a todo's completed state when its checkbox is clicked
- **File:** `todo-app/src/App.tsx`
- **Acceptance criteria:** After checking or unchecking a todo's checkbox, the corresponding todo's completed property in state toggles and the UI reflects a line‑through style on the todo text
- **Estimate:** 60 minutes

## Task 2: Update App.tsx to correctly delete a todo when the delete button is clicked
- **File:** `todo-app/src/App.tsx`
- **Acceptance criteria:** After clicking a todo's delete button, that todo is removed from the list and no longer appears in the DOM
- **Estimate:** 45 minutes

## Task 3: Update App.tsx to correctly filter the displayed todos by the selected filter tab
- **File:** `todo-app/src/App.tsx`
- **Acceptance criteria:** Selecting a filter tab updates the list so that only the todos matching the filter criteria (All, Active, or Completed) are rendered and all other todos are excluded
- **Estimate:** 60 minutes
